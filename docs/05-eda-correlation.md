# EDA · 출고 ↔ 외부변수 상관분석 (W5)

수요예측(W6)에 들어가기 전, 출고 실적과 외부 변수(기온·공휴일) 사이에
**예측에 쓸 만한 신호가 실제로 존재하는지** 확인하는 단계.

## 1. 분석 데이터

`analytics/forecast/data_prep.py`의 `build_features()`가 생성하는 피처 행렬:

| 컬럼 | 의미 | 출처 |
|------|------|------|
| `date`, `material_no` | 일자·자재 | 자재문서(MSEG) |
| `qty` | 일별 출고량 | 이동유형 방향 −1 집계 |
| `avg_temp`, `precip_mm` | 평균기온·강수 | ext_weather |
| `is_weekend`, `is_holiday` | 주말·공휴일 | 요일 파생 / ext_holiday |

핵심: **출고(이동유형 201)만 집계**하고 입고(101)는 제외한다. 출고가 곧 수요다.

## 2. 상관분석 결과 (기온 ↔ 출고, Pearson r)

| 자재 | r | 해석 |
|------|------|------|
| 전기히터·온수매트·핫팩 | **≈ −0.95** | 난방형 — 기온↓ 출고↑ |
| 선풍기·미니선풍기 | **≈ +0.95** | 냉방형 — 기온↑ 출고↑ |
| 생수 | ≈ +0.46 | 약한 양의 상관 |
| 물티슈 | ≈ −0.02 | 무관(대조군) |

→ 난방·냉방 품목에서 **강한 선형 관계**가 확인된다. 다중회귀가 잡아낼 신호가 충분하다.
→ 물티슈는 기온 무관 → 모든 품목에 날씨를 쓰면 안 되고, **품목 특성에 맞는 변수 선택**이 필요하다는 인사이트.

## 3. 추가 관찰

- 월별 추이에서 히터와 선풍기는 **정확히 반대 패턴**(겨울 히터↑ / 여름 선풍기↑).
- 주말·공휴일에는 전 품목 출고가 약 +25% 증가 → 회귀에 `is_weekend`, `is_holiday`를 포함한다.

## 4. W6 모델링 방향

- **종속변수**: 일별 출고량(`qty`)
- **독립변수**: `avg_temp`, `is_weekend`, `is_holiday` (+ 필요시 `precip_mm`)
- **모델**: 품목별 다중선형회귀(OLS). 계수 해석으로 "기온 −1℃ → 출고 +x개" 도출.
- 물티슈처럼 상관이 약한 품목은 이동평균 등 단순 기법으로 대체 검토.
